# 骨碌碌 Emoji / 颜文字悬浮球

为骨碌碌文字世界添加一个 Emoji / 颜文字悬浮球。

## 功能

- 页面右下角显示 Emoji 悬浮球
- 支持 Emoji 插入
- 支持颜文字插入
- 支持最近使用
- 支持自定义添加 Emoji
- 支持自定义添加颜文字
- 支持管理模式删除自定义内容

## 安装

请先安装 Tampermonkey 或 ScriptCat。

然后点击下面的链接安装：

[点击安装脚本](https://raw.githubusercontent.com/你的GitHub用户名/gululu-emoji-helper/main/gululu-emoji-helper.user.js)

## 使用方法

1. 打开骨碌碌页面。
2. 点击评论框、回复框或闲聊输入框。
3. 点击页面右下角的 😊 悬浮球。
4. 选择 Emoji 或颜文字。
5. 点击后会插入到当前输入框。

## 数据说明

自定义 Emoji、颜文字和最近使用记录保存在浏览器本地 localStorage 中。

不会上传到服务器。
